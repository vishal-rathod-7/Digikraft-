# DigiKraft

A simple, secure and powerful file management system.

---

DigiKraft is a lightweight, fast and secure web application that allows users to upload, manage and download their files safely. Built using **Flask (Python)** and **MySQL**, it focuses on simplicity, privacy and speed.

---

##  Key Features

* Secure Login & Registration
* Easy File Upload & Download
* Delete & Manage Documents
* Clean and Minimal Dashboard
* Fast API Performance
* Simple and Modern UI
